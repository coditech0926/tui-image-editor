# Image Editor
Canvas image editor

![image](https://cloud.githubusercontent.com/assets/18183560/20796021/86d53b88-b817-11e6-8fc3-2475d070d643.png)

## Feature
* Load image to canvas
* Undo/Redo (With shortcut)
* Crop
* Flip
* Rotation
* Free Drawing
* Line drawing
* Shape
* Icon
* Text
* Mask Filter
